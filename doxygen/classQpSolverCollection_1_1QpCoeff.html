Detailed Description

Class of QP coefficient.

Todo:
Support both-sided inequality constraints (i.e., d_{lower} <= C x <= d_{upper}) QLD, QuadProg, and NASOQ support only one-sided constraints, while LSSOL, JRLQP, QPOASES, and OSQP support both-sided constraints.
